# PhD Thesis: WPE Project

LaTeX source for University of Pittsburgh thesis proposal on multi-domain omnibot robots.

## Build
```bash
cd src
make clean
make
```

## Files
- `src/main.tex`: Main document.
- `src/classes/`: Pitt thesis class files.
- `src/bibs/`: Bibliography.
- `src/parts/`: Chapters.
- `src/images/`: Assets.

GitHub: https://github.com/darebear/phd-thesis
